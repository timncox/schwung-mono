01 / Start

Choose how much machine you want.

Mono is installed in two forms. Use the full-surface build to write complete six-track patterns, or load one voice into a normal Move track.

The complete instrument.

Mono takes over Move's surface and turns it into a six-track synth with its own 64-step sequencer and per-step parameter locks.

6 tracks64 stepsLocksLive keys
Launch MonoOpen the Schwung overtake module named mono.
Choose a trackPress one of the first six pads on the top row. Track 1 is selected at startup.
Choose a machinePress top-row pad 7 to move forward. Hold Shift while pressing it to move backward.
Play and sequencePlay the lower three pad rows, tap step keys to add trigs, then press top-row pad 8.

A machine inside Move.

Mono Voice behaves like a normal Move sound generator: Move owns the track, clips, pads, and sequencing while Mono supplies the synthesis engine.

1 voiceMove clips7 pages6 machines
Add Mono VoiceLoad mono-voice into a normal Move track.
Play normallyUse Move's pads and sequencer exactly as you would with another sound generator.
Pick a pageTurn the jog wheel, or use the left and right arrows, to move through SYNTH, AMP, FILTER, EFFECT, and LFO 1–3.
Pick a machine or deeper controlHold Shift and turn the jog wheel to change machine. Hold Shift while turning knobs 1–8 to edit that machine's secondary synthesis layer.

03 / Locks

Make every step its own sound.

A parameter lock stores a knob value on one step. The base sound stays untouched everywhere else.

01

Hold a step.

Keep any of the 16 step keys pressed. Use the arrows first if the step lives on page 2, 3, or 4.

02

Turn a knob.

Mono writes that value only to the held step. The LED turns purple to show the step now carries a lock.

03

Remove it.

Hold the step, hold Shift, then turn the same knob. That parameter returns to the track's base value.

Red · trig
Purple · lock
White · playhead
Green · held
Live automationPress Move's Record button, start playback, then turn Mono controls. Full Mono writes the selected track's current step; Mono Voice follows Move's clock while Move keeps recording the notes. Continuous values glide over about 30 ms, while waveform, LFO routing, trigger, and wave choices stay stepped.
04 / Save

Keep the sound. Keep the sequence.

Choose the save that matches where the sequence lives. Mono presets and Move Sets preserve different layers of the instrument.

VOICE

Save the sound.

In Schwung's Chain Editor, highlight the Mono Voice synth block, hold Shift and click the jog wheel, then open User Presets → Save current. This recalls the machine, every sound parameter, and Mono's live automation locks.

SET

Save sound + Move clips.

For notes recorded in Move's normal sequencer, save the whole Move Set: hold Shift and press Step 1. Sets save automatically and contain the clips, track state, and Mono Voice automation.

MONO

Save the six-track pattern.

Inside full-surface Mono, hold Shift and click the jog wheel. Choose Save current, name it, and confirm. Recall it from the same list; all six tracks, 64 steps, and parameter locks return.

Eight knobs. Seven views. No menu maze.

Use this live panel to see every label exactly as it appears on Move. Change machine or page; move the sliders to update the mock display.

Stacked detuned saws with sub oscillators for wide basses, leads, and drones.
Synthesis machine
Shift synthesisOn the SYNTH page, hold Shift and turn any parameter knob for eight additional machine controls. The first four change with the machine; knobs 5–8 are Drift, Fold, Bits, and Noise. Each LFO can target Pitch or any of the 64 sound parameters—including settings on the other LFOs—and shows the destination by name. Trigger and Wave step through five named choices instead of raw numbers.
06 / Machines

Six starting points. Six different instincts.

M–01SUPERWAVE

SW Saw

Detuned saw layers and sub oscillators. Start here for broad bass, bright chords, and unstable unison.

  • Unison
  • Sub mix
  • Shift: bright / stack / sub
M–02SUPERWAVE

SW Pulse

A unison pulse stack with animated width. Good for hollow bass, nasal leads, and moving PWM beds.

  • Pulse width
  • PWM
  • Shift: asym / sync / sub
M–03SUPERWAVE

SW Ensemble

Four pitched oscillators with waveform, pulse width, and stereo chorus controls. The harmony machine.

  • 4 pitches
  • Wave
  • Shift: four voice levels
M–04DIGITAL CHIP

SID 6581

Chip-style digital oscillator with selectable waveform, modulation source, and hard-edged frequency interaction.

  • Wave
  • Mod source
  • Shift: ring / sub / chaos
M–05WAVETABLE

DigiPRO

A factory-independent 512-sample, 12-bit wavetable voice with position modulation and oscillator sync.

  • Wave position
  • Mod
  • Shift: second wavetable
M–06FM+

FM+ Static

Two-block FM with ratios, fine tune, feedback, envelope depth, modulator level, and tone.

  • Feedback
  • Shift: third operator
07 / Reference

Everything worth remembering.

Mono / full surface

Top pads 1–6Select one of six synth tracks.
Top pad 7Next machine. Shift + pad selects the previous machine.
Top pad 8Start or stop Mono's internal sequencer.
Lower 24 padsChromatic keyboard for the selected track, starting at MIDI note 48.
Remote audition keysPlay MIDI notes 48–60 on the selected Mono track. The large label is the note; the small A–K label is its optional computer-key shortcut. A click is held long enough to be audible. If the Remote UI says it is waiting, select another Slot tab and return to reconnect it.
Step keysTap to toggle trigs; hold while turning a knob to lock it.
ArrowsSelect steps 1–16, 17–32, 33–48, or 49–64.
Jog wheelMove through the seven parameter pages.
Knobs 1–8Edit the current page. Touching either group of four focuses that clearer display bank.
Shift + knobsOn SYNTH, edit the selected machine's secondary eight-control layer.
RecordArm live parameter-lock capture. Start the internal sequencer and turn knobs; continuous values glide between recorded steps.

08 / Install

Two modules. One archive each.

Already on Tim's Move.

The current development build is installed as mono and mono-voice. After a new deployment, rescan Schwung modules or restart Move so the new files are loaded.

For another Move, install Schwung, then upload mono-module.tar.gz and/or mono-voice-module.tar.gz with schwung-manager.

Which archive?

mono-module.tar.gz
Full-surface, six-track instrument.

mono-voice-module.tar.gz
Single sound generator for a normal Move track.

Source and public release publishing are the next project step.
